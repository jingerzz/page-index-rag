# Roadmap

## Near-term (next releases)

- Harden SEC HTML/XHTML parser handling.
- Add integration tests for fetch → ingest → MCP retrieval flow.
- Stabilize PageIndex engine APIs.
- Improve index metadata quality and consistency.

## Mid-term

- Evaluate split/packaging strategy for FidoSEC CLI.
- Publish structural retrieval benchmarks vs vector baselines.
- Add optional hybrid retrieval layer while preserving vectorless mode.

## Long-term

- Multi-domain ingestion templates beyond SEC filings.
- Retrieval quality and hallucination-reduction evaluation suite.
- Managed deployment profile for productizable research workflows.
